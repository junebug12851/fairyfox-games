## v0.25.3 — 2026-07-23 — GROW

**Symmetry gets "depth inside the mechanic" (the 10th game on the layer).** Symmetry (planted
2026-07-04) was the oldest game still missing the layer, with the exact plateau the sweep flags:
`fallSpeedOf` was keyed only on the stage index and the stages stop at Singularity (score 72) —
past that the fall speed was flat forever. All four depth items land on the one *spread* verb —
no new control, all safe to not know:

- **The FACET (the hidden tech).** A catch already succeeds anywhere within the forgiving
  `CATCH` 0.13 of the lane; inside it hides a razor `FACET_BAND` 0.022 — land the catcher
  **dead-on the lane** and the catch is a *facet*: +`FACET_BONUS` 2 on top of the point, a gold
  bloom, and a streak. Taught nowhere; a loose catch scores exactly as it always did and silently
  breaks the chain (a miss breaks it too). A dead-on **twin** is two facets in one tick, so the
  rewarding twin cadences (Reflection, Kaleidoscope's twins) are the quiet greed window.
- **RADIANCE (the reversal it unlocks).** `RAD_TRIGGER` 3 facets in a row light the mirror: for
  `RAD_TICKS` 300 (~5s) **every point doubles** — catches, facet bonuses and twin bonuses — with
  the catchers and catch line burning gold (colour only; geometry, the catch window and the maths
  are untouched; reduced-motion safe). The triggering catch is never doubled, and the streak
  resets so the window must be re-earned. The precise hand becomes the greedy one.
- **No plateau.** `fallSpeedOf` now rides a smooth score asymptote (`fallScale` ×1 → ×1.5,
  half-travelled at score `FALL_K` 90) on top of the stage steps, hard-capped at `FALL_HARD_MAX`
  0.031 — set *above* the asymptote's own limit, so the cap is a safety bound for rogue overrides,
  never a felt plateau. Regression-pinned still climbing at score 600, override-proof.
- **A secret stage: Infinity.** A sixth stage at score 150, `secret: true`, past Singularity —
  printed on no start screen (the tip's stage ladder and cadence list were removed, replaced with
  a curiosity hook) and announced exactly once, by being reached (gold toast + badge).

3 new skill-safe badges (9 → 12: Facet · Radiant · Through the glass), `totals.facets` (absent in
legacy metas — upgrades losslessly), run report gains `· N facets`, `TickResult` gains
`{facets, radiance}`. Also applied Poise's two v0.25.2 game-over fixes to Symmetry while here:
the invisible-headline `background-clip` bug and the uncapped badge column (now `34vh` +
thin scrollbar). +12 pure-core tests (31 → **43**); collection **730/730** green. **Depth-layer
rollout: 10 of 13** (left: Arc and Sluice).

## v0.25.2 — 2026-07-22 — GROW

**Poise gets "depth inside the mechanic" (the 9th game on the layer).** Poise was the oldest
game still missing it and already carried one of the four items (the no-plateau gravity
asymptote, v0.22.2), so this run added the other three on the one *tilt* verb — no new control,
nothing bolted on.

- **The STILL (the hidden tech).** A catch is a *still* when the ball arrives essentially at
  rest (`STILL_VEL` 0.006 — ~14% of the base terminal roll speed, and far less late on) **and**
  the approach genuinely peaked at `STILL_PEAK` 0.010 first. That second clause is the anti-farm
  and the whole design: you can't creep the beam at a crawl and collect: you have to *carry speed
  and kill it dead on the mark*. It pays `STILL_BONUS` 2 on top of the catch, blooms gold, and
  builds a streak. Taught nowhere — neither bound is drawn. Tuned against a bot: a naive chaser
  lands **0** stills in hundreds of catches (so it never fires by accident, and a first-timer's
  game is untouched); a deliberate braker lands ~13%.
- **EQUILIBRIUM (the reversal it unlocks).** `EQ_TRIGGER` 3 stills in a row settle the beam:
  for `EQ_TICKS` 300 (~5s) **every point doubles**, beam and target burning gold (colour only —
  geometry, physics and the catch window are untouched). The triggering catch is never doubled.
  The calm, braked line — the game's namesake — is quietly the greedy one.
- **A secret stage: The Eye.** A sixth stage at score 120, `secret: true`, past the Tempest.
  It is printed on no start screen (the tip's stage ladder was removed) and announces itself
  exactly once, by being reached.

**Score vs catches.** Because a still pays a bonus, `score` is now *points* and the new `catches`
counts targets collected; the three "catch N in a run" badges test `catches`, so a bonus can never
inflate them. `nearMissLine` speaks in points to match. 3 new skill-safe badges (9 → 12) and
`totals.stills` (absent in legacy metas — upgrades losslessly). `tryCatch`/`tick` now return rich
results (`{caught, still, points, eqLit}`) so the shell can bloom the catch.

**Two bugs the preview caught, both fixed in Poise.** (a) The game-over headline rendered as a
**solid gradient bar with invisible text**: `.title` sets `background-clip:text`, but
`#gameover .title` / `.title.record` use the `background` **shorthand**, which resets
`background-clip`. **All 16 games have this** — fixed here, logged in `plans/next-steps.md` for a
housekeeping run (two lines per game, but 15 more game-over screens to eyeball first). (b) With
12 badges, a big run's badge column overflowed short viewports and pushed the score off the top;
capped at `34vh` with a thin scrollbar.

+14 pure-core tests (42 → **56**); collection **718/718** green. Previewed in Chrome from the
built `_site/` — Still, Equilibrium, The Eye, the game-over card and the landing page. Released `dev → main` by default on green (PATCH). **Depth-layer rollout: 9 of 13.**

## v0.25.1 — 2026-07-21 — HOTFIX

**Every game showed "Couldn't load" on the live site — fixed.** The shared coins module lived in
`games/_shared/`, and **Jekyll excludes underscore-prefixed folders from its build output**, so
`_site/games/_shared/` was never deployed. On the live site each game's ES-module import of
`../_shared/coins-game.js` 404'd, the module failed to load, and every game (all 16, including the
new Drove) fell back to its "Couldn't load" screen. Local previews had served the **raw repo**
(where `_shared/` exists), so it looked fine — the bug only surfaced in the built `_site/`. **Fix:**
renamed `games/_shared/` → `games/shared/` (no underscore, so Jekyll copies it through) and updated
the import in every game shell. Verified by building the site and booting games from the **built
`_site/`** — Polarity, Sluice, Reprise, and Drove all load with their coin controls, and
`_site/games/shared/coins-game.js` is present. **704/704** tests green; check-links clean. Lesson:
a shared asset used by the passthrough games must never sit under a `_`-prefixed path.

## v0.21.0 — 2026-07-11

**PLANT: new game Tether — a genuinely new verb (swing/grapple).** The collection's first
pendulum game, and its 12th. Anchors hang ahead of you across an endless sky; **hold** to rope
onto the nearest reachable one and swing beneath it on a real pendulum, **release** to fly. Miss
the next anchor and you fall past the floor. One control: rope on, rope off.

**The hook — the whip.** Exit velocity is the swing's *tangential* velocity, so the release angle
**is** the launch angle, which turns letting go into a pure projectile trade-off: release near the
bottom and you fly fast but **flat** into the ground; hang on to the top and you're high but
**stalled**. The sweet spot is the classic ~45° launch, partway up the forward swing. A whip grows
the multiplier (×2…×9) **and boosts the launch**, so it is not merely points — it is the distance
that clears the next gap. Skill and survival are the same act. Staying on the rope **pumps** the
swing higher (hold to wind up, let go to launch).

Ships on **varied structure + the full growth architecture from birth**: 6 stage-gated
`FORMATIONS` (Steady/Rise calm; Stagger, The Chasm, Canopy, The Gauntlet notable + `minStage`
gated), a stage arc (Sway → Momentum → Airborne → Freeflight → Skybreak), a smooth gap asymptote
(no plateau), and meta (`tether.meta`, 13 skill-safe badges, run-report + near-miss). **Depth
inside the one verb:** a hidden **snap** sub-window straddling the true 45° optimum (the whip arc
is drawn on screen; the snap window inside it is not), **Slipstream** (a snap streak earns a
double-score window), and a **secret Zenith stage**. Pure core + **32 tests**; collection
**435/435** green.

**Two bugs nearly killed it, both now pinned by regression tests.** (1) A flat angular-speed clamp
let a fast catch on a long rope carry enough energy to swing *over the top* of the anchor, after
which the angle ran away unbounded (θ ≈ −28 rad) — the cap is now on **energy**, so amplitude, and
therefore the angle, is bounded by construction. (2) Applying that cap *outside* the cap angle
pinned angular velocity at zero every tick and **froze the pendulum solid** in mid-air. Both
presented identically from the outside: a run that was alive but going nowhere. Earlier still, a
*lossy* rope catch (discarding the radial component, as a real rope would) bled energy out of every
run until the swing decayed below the whip window and the player was stranded — the catch now
conserves speed, which is what lets a run flow. The soak test asserts the real invariant: **if a
run is still breathing, it's because it's thriving, not because it's stuck.**

**New game: Sluice — a colour-sorting reflex game (a genuinely new verb: *sort/route*).**
Coloured sparks fall one at a time down a central chute; the player routes each into the
**channel** that matches its colour before it lands (press a channel's number key **1–4**, or
tap it). The single decision is *which channel* — a categorise-and-route, distinct from every
existing verb (steer, time-a-catch, thrust, flip-match, aim-bounce, precision-stack, keep-aloft,

a spark's colour never moves but its matching channel keeps sliding to a new slot — you read the
row, not a fixed key. Core-fun hook = a **snap combo**: routing a spark *early* (within the first
`SNAP_FRAC` of its fall) grows a ×2…×9 multiplier; a slow-but-correct route scores without
growing it, so each spark is a fast-read-vs-sure-read gamble. Wrong channel or a spark that lands
unrouted is a miss; three misses end the run.

fns, injectable seedable RNG, no DOM/canvas/timers): `route`/`tick`/`spawnDrop`/`resolveDrop`, a

*sequence of named formations* — **Steady**, **Run**, **Alternate** (calm), **Shuffle**,
**Cascade**, **Rush**, **The Churn** (notable) — pulled from a stage-weighted `FORMATIONS` pool
(`pickFormation`/`loadFormation`), gated by `minStage` so climbing the stages introduces the
