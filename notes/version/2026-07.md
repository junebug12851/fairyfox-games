## v0.18.0 — 2026-07-06

**New game: Sluice — a colour-sorting reflex game (a genuinely new verb: *sort/route*).**
Coloured sparks fall one at a time down a central chute; the player routes each into the
**channel** that matches its colour before it lands (press a channel's number key **1–4**, or
tap it). The single decision is *which channel* — a categorise-and-route, distinct from every
existing verb (steer, time-a-catch, thrust, flip-match, aim-bounce, precision-stack, keep-aloft,
balance, mirror-coordination, judge-power). The signature twist: the channels **rearrange**, so
a spark's colour never moves but its matching channel keeps sliding to a new slot — you read the
row, not a fixed key. Core-fun hook = a **snap combo**: routing a spark *early* (within the first
`SNAP_FRAC` of its fall) grows a ×2…×9 multiplier; a slow-but-correct route scores without
growing it, so each spark is a fast-read-vs-sure-read gamble. Wrong channel or a spark that lands
unrouted is a miss; three misses end the run.

Built to the full quality bar from day one. **Pure core** (`sluice.core.js` — plain data + pure
fns, injectable seedable RNG, no DOM/canvas/timers): `route`/`tick`/`spawnDrop`/`resolveDrop`, a
`bins` permutation with a guaranteed-different `permuteBins`, and the whole progression stack.
**Varied structure + progression** per `reference/varied-structure.md`: a run is a seeded
*sequence of named formations* — **Steady**, **Run**, **Alternate** (calm), **Shuffle**,
**Cascade**, **Rush**, **The Churn** (notable) — pulled from a stage-weighted `FORMATIONS` pool
(`pickFormation`/`loadFormation`), gated by `minStage` so climbing the stages introduces the
meaner patterns (The Churn is the stage-3 crescendo). **Stages** (Trickle → Stream → Rapids →
Cataract → Maelstrom) both speed the fall *and widen the sort* (channels grow 3 → 4 by stage —
`binsAt`). **Meta** (`sluice.meta`: lifetime sorts/snaps + bestStage/bestMult + 10 skill-safe
badges, run-report card; legacy `sluice.best` preserved). External shell module + classic-script
boot-failure fallback, guarded render loop, global error/rejection logging, reduced-motion-safe
formation cue. **+35 pure-core tests** (routing/resolution, snap-combo scoring, lives/death,
stage ladder, formation pool well-formed + stage-gated + deterministic + distinct-seeds→distinct
structure + channels always re-shuffle, meta reducer). Wired into the root README table, a landing
card (`Sorting`/`Reflex` chips, `data-updated` today), masthead **Games 11**, and a player-facing
`kind:"new"` changelog entry. Collection **361/361** green. Released `dev → main` by default on
green (MINOR). **Rollout unchanged: 3 of 11 games on varied structure (Polarity, Echo Chamber, Ink
Bloom); Sluice ships on the pattern from birth, so 4 of 11 carry it.**
