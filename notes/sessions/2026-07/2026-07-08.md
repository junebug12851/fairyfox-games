## GROW — Orbit Slingshot onto varied structure (6th game on the pattern) · v0.19.4

**What.** Brought Orbit Slingshot onto the varied-structure + progression pattern — it was the
lowest-coverage game (Growth-Architecture layers 1–2 only, a flat one-target-at-a-time spawn).
Its `pickTarget` drew one random point in the annulus each pickup, so every run traced the same
shapeless sweep. Now a run is a seeded **sequence of named target formations**, copied in shape
from Polarity/Ink Bloom into Orbit Slingshot's own core.

**How (core).**

- Added `CONFIG.FORMATIONS` — 6 patterns as plain data (`id`, `name`, `minStage`, `notable`,
  `weight(stage)`, `build`): **Belt** (0, calm), **Cluster** (0, calm), **Ring** (0, notable),
  **Ladder** (1, notable), **Perihelion** (2, notable), **Swarm** (2, notable).
- `build(ctx)` returns `{ang, rFrac}` specs — `ang` an absolute bearing, `rFrac` a 0..1 position
  across the **current stage-tightened annulus** (`targetAnnulus`), so the existing per-stage
  inward pull + `targetRadius` shrink still layer on top of the pattern. All pure given `ctx.rng`.
- `pickFormation(cfg, stage, rng, prevId)` — seeded, stage-weighted pick over the eligible pool,
  softly avoiding an immediate repeat (identical to the reference). `loadFormation(g)` fills
  `g.formTargets`, marks the head spec, records `formId`/`formName`/`formNotable`.
- Refactored `pickTarget` to pull one spec from the queue (loading a fresh formation when spent);
  the placed target carries `form` + `formHead` (notable-only). `reset` seeds `formTargets:[]` /
  `formId:null` so the first placement pulls a formation (frame-one-safe; the opening pool is
  calm). `tick` now surfaces a `formation` cue when a notable head target appears.

**How (shell/HTML).** Added a `#formCue` element + peripheral flash CSS (reduced-motion honoured);
shell shows it on `r.formation`, clears on death. Start-panel tips name the pattern set
(Belt · Cluster · Ring · Ladder · Perihelion · Swarm).

`targetAnnulus` inner-edge pull; distinct seeds → distinct sequences, same seed reproduces;
queue never starves over a long run; `tick` cues only notable heads; frame-one regression with a
formation loaded. Two existing no-op `tick` deepEquals updated for the new `formation:null` field.
