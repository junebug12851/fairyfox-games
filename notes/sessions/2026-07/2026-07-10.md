## GROW MILESTONE — "depth inside the mechanic": Polarity reference build · v0.20.0 · ✅ released

**Why.** Owner playtest feedback, verbatim: the games are colorful, fun, interactive — "but then I
quickly get bored… after 5 minutes it's 'ah, it's just one of those games'… you keep mentioning
progression and stages but I don't see it." Also: the intro walls of text get clicked past; the fun is
learning by intuition.

**Diagnosis (grounded in the code, not theory).** Read Polarity's core + shell. The collection chases
depth with two tools that *by their own design notes* don't create the *feeling* of depth: **meta**
(badges/lifetime totals — invisible on a fresh play) and **varied structure** (formation shuffling — but
every formation is a *fixed intensity*, and the hardest unlocks a third of the way in). Meanwhile the one
felt axis — speed — **plateaued**: `speedOf` hit `SPEED_MAX` near ~100 gates and went flat forever. So
after five minutes a player has seen the whole ceiling. Wrote it up as a new standard,
`reference/depth-inside-the-mechanic.md` (depth ≠ complexity; the five-tool toolkit; the extra checklist),
plus the concrete plan `plans/2026-07-10-depth-inside-the-mechanic.md`. Owner asked to save the research +
understanding to notes and lean on notes heavily — done.

**What shipped (Polarity, all on the single flip verb — no new controls, all safe-to-not-know):**

- **No plateau.** `speedOf` is now a smooth asymptote `BASE + (CAP−BASE)·c/(c+K)` — always still creeping
  up within a human run, passing the old cap around gate ~198 and heading toward 12. A regression test
  pins it *rising past the old 100-gate point*.
- **Snap** (the hidden tech). A tight inner window `SNAP_TICKS=4` inside the precise window: a razor-close
  flip pays a flat `SNAP_BONUS` and builds `snapStreak`. Never mentioned on the start screen — discovered
  by cutting flips close. Raises the skill ceiling (there's always a tighter flip worth more).
- **Overcharge** (the earned surprise). `OC_STREAK=6` consecutive snaps → a ~5s window where every gate
  **scores double** and the field blooms gold. The Pac-Man power-pellet beat; re-earned each time.
- **Supernova** (the face-down card). A **secret 6th stage** past Singularity (`at:260`), unnamed on the
  start panel; reaching it fires a reveal banner + a badge.
- **Intro trim.** The two-paragraph manual → one line ("Match the gate's colour to pass through. Flip at
  the last instant to build your multiplier.") + a curiosity hook ("The deeper you go, the more there is
  to find.").
- Four new skill-safe badges (`snap`/`razor`/`overcharge`/`supernova`); `totals.snaps` added to meta
  (legacy blobs upgrade losslessly). `TickResult` gains `snap`/`overcharge`.

**Verification.** +8 net pure-core tests; collection **392/392** green (scoped `games/**`). Chrome-
previewed **live** on a local Node static server: trimmed start panel; a running game (DRIFT chip, ×1
readout, orb+gates clean); forced Overcharge via a temporary debug hook (gold **⚡×10** readout, golden
field bloom, orb halo — RIPTIDE chip); the **Supernova** reveal banner + gold stage chip + shockwave.
No console errors anywhere. Debug hook removed and grep-verified gone before shipping.

**Ship.** VERSION → `0.20.0` (MINOR — new play layer + reference build); player changelog entry
(`_data/changelog.json`, written to *invite discovery*, not spell out the secrets); `_games/polarity.md`
date + tagline; README re-gen (`npm run gen:readme`, 11 games). `growth-roadmap.md` priority lever
updated: **depth-inside-the-mechanic now leads** (supersedes "add one more formation"). Released
`dev → main` (MINOR via `release/0.20.0`), tagged `v0.20.0`, back-merged.
