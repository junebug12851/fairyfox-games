## v0.25.3 — Symmetry: the FACET, RADIANCE, the fall asymptote, and Infinity

**Why Symmetry.** Three games still lacked the depth layer — Symmetry, Arc and Sluice.
Symmetry (born 2026-07-04) is the **oldest** of them, and it had the exact plateau the
sweep flags: `fallSpeedOf` keyed only on the stage index, stages stopping at Singularity
(score 72) — past that the one felt axis was flat forever. Plan:
`plans/2026-07-23-symmetry-depth.md`.

**What landed (all on the one *spread* verb, taught nowhere, safe to not know):**

- **The FACET** — a razor `FACET_BAND` 0.022 dead-on sub-window inside the forgiving
  `CATCH` 0.13. A dead-on catch pays +`FACET_BONUS` 2, blooms gold, builds a streak; a
  loose catch scores as it always did and silently breaks the chain (a miss too). A
  dead-on twin is two facets in one tick — the twin cadences become the quiet greed
  window, which folds neatly into the existing Reflection/Kaleidoscope structure.
- **RADIANCE** — `RAD_TRIGGER` 3 facets in a row → `RAD_TICKS` 300 (~5s) in which every
  point doubles (catches, facet bonuses, twin bonuses). The trigger tick is never
  doubled (`radActive` is read before any scoring); the streak resets so the window is
  re-earned. Shell: gold toast announced only when earned; catchers + catch line burn
  gold, colour-only, reduced-motion safe.
- **No plateau** — `fallScale` (×1 → ×1.5 asymptote, half at score 90) multiplies the
  stage-stepped fall speed, hard-capped `FALL_HARD_MAX` 0.031 *above* the asymptote's
  own limit (a safety bound for rogue overrides, never a felt plateau).
  Regression-pinned still climbing at score 600; override-proof pinned.
- **Infinity** — a secret sixth stage at score 150 (`secret: true`, gold tint), on no
  start screen; the tips' stage ladder + cadence list were removed for a curiosity hook.
  Reveal = gold toast + shake on entry, plus a badge.

**Meta:** 3 new skill-safe badges (9 → 12: Facet · Radiant · Through the glass);
`totals.facets` lossless legacy upgrade; run report gains `· N facets`; `TickResult`
gains `{facets, radiance}`.
