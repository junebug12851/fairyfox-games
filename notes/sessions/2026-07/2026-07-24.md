## v0.25.4 — Arc: the PIN, ONSLAUGHT, the pad asymptote, and Pinhole

**Why Arc.** Two games still lacked the depth layer — Arc and Sluice. Arc (born 2026-07-09)
is the older of the two, and it had the exact plateau the sweep flags: the pad half-width
was a per-stage constant and the stages stop at Dead-eye (land 42) — past that the pad never
got finer again, so the whole ceiling was visible in ~2 min. Ricochet was the template (also
a turn-based, discrete-shot game, so its Dead-Centre/Blaze shapes port cleanly to the lob).

**The four items, all on the one charge-and-release verb, all safe to not know:**

- **The PIN (hidden tech).** `pinBandFor(cfg, hw)` = `max(PIN_ABS 3, hw × PIN_FRAC 0.16)` — a
  razor sub-window inside the drawn bullseye band (`BULLSEYE_FRAC` 0.34). A land within it of
  dead centre is a pin: +`PIN_BONUS` 2 over the bullseye, gold burst, a streak. A fraction of the
  pad (not an absolute px) so it self-scales as the pad shrinks and stays clearly tighter than the
  bullseye on every stage (checked in tests). Off-centre bullseye / miss silently breaks it.
- **ONSLAUGHT (the reversal).** `PIN_TRIGGER` 3 pins in a row → `g.onslaught = ONSLAUGHT_LANDS 3`;
  the next 3 scoring lands pay ×`ONSLAUGHT_MULT` 2. The triggering land is scored *before* the
  streak check, so it's never itself doubled; a miss breaks the pin streak but does **not** consume
  a lit window (the lost life already punishes it). Shell: pad centre band + projectile halo burn
  gold (colour-only; reduced-motion honoured), a "Pin! +2" / "Onslaught ×2!" toast.
- **No plateau.** `padHalfWidth(cfg, landed)` = stage hw × `(1 − HW_SHRINK_SPAN 0.28 · landed/(landed
  + HW_SHRINK_K 70))`, hard-floored at `HW_HARD_MIN` 20px. Used in `spawnTarget` for both the pad
  width and the placement window (so pads still fit). ×1 at land 0 (opening feel unchanged); keeps
  shrinking within a stage, so the old Dead-eye flat-line is gone. `lob` reads the stored `g.target.hw`,
  so bullseye + pin both use the shrunk width consistently.
- **Secret stage Pinhole.** A 6th STAGES entry at land 66, `secret: true`, hw 26 (< Dead-eye 32, so
  the "pads shrink each stage" test still holds), announced once on reach (gold toast). The start
  tip's stage ladder + formation-name list were removed and replaced with a curiosity hook, so the
  secret isn't spoiled.
