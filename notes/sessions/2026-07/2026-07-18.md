## PLANT run — new game **Ward** (a genuinely new verb: guard / deflect) (v0.24.0)

**Verb pick:** every existing verb is offence or navigation (steer, time-a-catch, thrust,
flip-match, aim-bounce, stack, keep-aloft, balance, mirror, charge, swing/grapple, pour,
recall, sort). None is **defence** — guarding a point. Ward is the collection's first
**radial guard**: shards converge on a central core from the rim; you orbit **one shield**
around it and cover the threat. Distinct from Symmetry (mirror dual-catch on a line) and
Poise (balance) — the unit here is an *angle you point*, with a turn-rate cap that makes
opposite threats a real read.

**Built to the full bar (copied Polarity's shape into Ward's own core):**

- **Pure `ward.core.js`** — normalised unit-disk geometry (r=1 rim → 0 core), injected
  seedable RNG, no DOM/canvas/timers. `setAim` is the whole control surface; `tick` slews
  the shield toward the aim (capped `TURN_RATE`), spawns, moves shards inward, resolves a
  shard the first tick it crosses `SHIELD_R`.
- **Core-fun hook = the parry.** A block within `SHIELD_HALF` scores; a block within the
  tight inner `PARRY_HALF` is a **parry** → `mult` ×2…×9 + a flat bonus + reflect; a loose
  block breaks the combo to ×1; a shard crossing outside the arc slips through and strikes
  the core (a life; 3 strikes = dead). The Polarity precise/safe tension, re-expressed
  radially — safe is cheap, the point is greedy.
- **Depth from birth:** a **Surge** (5 parries in a row → ~5 s every-point-doubles window)
  and a **secret Aegis stage** past Citadel (score 200, revealed only on reaching it).
- **Varied structure + progression from birth:** stage-gated formations
  (`FORMATIONS`/`pickFormation`/`loadFormation`; `spawnShard` pulls each shard from the
  volley queue): **Drift** (calm) · **Fan** (a sweep) · **Salvo** (same-angle burst — the
  greed window) · **Pincer** (staggered near-opposite pairs — the two-sided read) ·
  **Scatter** · **The Siege** (crescendo). Stages Picket → Vigil → Rampart → Bastion →
  Citadel + secret Aegis; no-plateau speed asymptote. Meta `ward.meta` (lifetime
  blocks/points/parries, best stage/mult, **13 badges**), legacy `ward.best` preserved.
- **Shell** = external module + classic-script boot fallback, guarded loop, global
  error/rejection logging, reduced-motion-safe cues, mouse/touch/arrow-key aim, lives pips,
  parry ring, Surge bloom.

**Tests:** `ward.core.test.js` — **49** pure-core assertions (geometry helpers,

## GROW run — Ricochet gets "depth inside the mechanic" (v0.23.3)

**Target pick:** depth-layer rollout stood at 5/13 (Polarity, Brim, Echo Chamber, Ink Bloom,
Orbit Slingshot; Tether + Reprise born with it). Ricochet is the **oldest game without the
layer** (5th built) and had the sweep's exact plateau: `targetRadius` linear shrink
hard-floors at 12 px around score ~55; stages stop at Bank master (140). Plan written first:
`notes/plans/2026-07-18-ricochet-depth.md`.

**What shipped (all on the one aim-and-fire verb, all safe to not know):**

- **No plateau** — `targetRadius` = the old linear ramp × a smooth score asymptote
  (`R_SHRINK_SPAN` 0.18, knee `R_SHRINK_K` 150), hard-floored at `R_HARD_MIN` 9 px.
  Strictly tightening past the old floor (regression-pinned 60 → 600), override-proof.
- **The Dead Centre (hidden tech)** — `computeShot` now tracks each target's closest
  approach; a collected target threaded within `PIN_BAND` (4 px) of its centre is a dead
  centre: `PIN_BONUS` +2 on top of the bank, a gold burst, a quiet gold `#pinCue` line, and
  a streak. Taught nowhere; an off-centre collect or a miss breaks it.
- **The Blaze (the reversal)** — 3 dead centres in a row → the next 2 scoring shots pay
  double (bank + pin bonuses). Triggering shot never doubled (Orbit's aurora shape); a miss
  doesn't consume the window; gold comet/launcher is colour-only.
- **Secret Legend stage** — score 240, `secret: true`, reveal toast + badge; start tips
  trimmed to a curiosity hook (stage ladder + layout enumeration dropped).
- **Meta:** `totals.pins` (lossless legacy upgrade), RunSummary + run report gain
  pins/blazes, 3 new skill-safe badges (8 → 11): Dead centre, Blaze, Legend.
